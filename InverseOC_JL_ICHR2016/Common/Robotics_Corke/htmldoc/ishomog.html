ishomog

PURPOSE

ISHOMOG Test if argument is a homogeneous transformation

SYNOPSIS

function h = ishomog(tr)

DESCRIPTION 

ISHOMOG Test if argument is a homogeneous transformation

    H = ISHOMOG(TR)

  Returns true (1) if the argument tr is of dimension 4x4.
